# Warehouse Management System - Backend API

## Overview
Node.js + Express.js REST API dengan MySQL database dan JWT authentication untuk sistem manajemen gudang.

## Tech Stack
- **Runtime**: Node.js 18+
- **Framework**: Express.js 4.x
- **Database**: MySQL 8.0
- **Auth**: JWT + bcrypt
- **Validation**: Joi
- **Docs**: Swagger/OpenAPI 3.0
- **Testing**: Jest + Supertest

- ### 1. Prerequisites
- Node.js 18+
- MySQL 8.0+

### 2. Installation
```bash
# Clone repository
git clone <repo-url>

# Install dependencies
npm install

# Run
npm run dev
